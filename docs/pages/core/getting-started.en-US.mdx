# Getting Started

## Installation

Install `@wagmi/core` and its ethers peer dependency.

{/* prettier-ignore-start */}
<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab>
```bash
npm i @wagmi/core ethers
```
  </Tab>
  <Tab>
```bash
pnpm i @wagmi/core ethers
```
  </Tab>
  <Tab>
```bash
yarn add @wagmi/core ethers
```
  </Tab>
</Tabs>
{/* prettier-ignore-end */}
