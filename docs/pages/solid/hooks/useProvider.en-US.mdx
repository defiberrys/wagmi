# useProvider

Solid hook for accessing `Client`'s ethers Provider.

```ts
import { useProvider } from '@wagmi/solid'
```

## Usage

```tsx
import { useProvider } from '@wagmi/solid'

function App() {
  const provider = useProvider()
}
```

## Return Value

```tsx
Accessor<Provider>
```

## Configuration

### chainId (optional) (signal)

Force a specific chain id. The wagmi `Client`'s ethers `provider` must be set up as a [chain-aware function](/react/client#provider-optional) for this to work correctly.

```tsx {5}
import { useProvider } from '@wagmi/solid'

function App() {
  const provider = useProvider({
    chainId: () => 1,
  })
}
```
