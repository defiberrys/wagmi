# Fallback

The `fallbackProvider` configures the chains with the default RPC URL specified on the `chain` object and also provides an [ethers.js `StaticJsonRpcProvider`](https://docs.ethers.io/v5/api/providers/jsonrpc-provider/#StaticJsonRpcProvider).

```ts
import { fallbackProvider } from 'wagmi/apiProviders/jsonRpc'
```

## Usage

```ts
import { configureChains, chain } from 'wagmi'
import { fallbackProvider } from 'wagmi/apiProviders/fallback'

const { chains, provider } = configureChains(
  [chain.mainnet, chain.polygon],
  [fallbackProvider()],
)
```

<Callout emoji="⚠️">
  Only having `fallbackProvider` in your API providers will make the chain use
  the public RPC URL which could lead to rate-limiting. It is recommended to
  also include another API provider in your list (such as: `alchemyProvider`,
  `infuraProvider` or `jsonRpcProvider`).
</Callout>

## Return Value

```ts
{
  chains: Chain[],
  provider: StaticJsonRpcProvider,
  webSocketProvider: WebSocketProvider
}
```
