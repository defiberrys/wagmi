# Default

The `defaultProvider` configures the chains with a public RPC URL and also provides an [ethers.js `getDefaultProvider`](https://docs.ethers.io/v5/api/providers/#providers-getDefaultProvider).

```ts
import { defaultProvider } from 'wagmi/apiProviders/default'
```

## Usage

```ts
import { configureChains, chain } from 'wagmi'
import { defaultProvider } from 'wagmi/apiProviders/default'

const { chains, provider } = configureChains(
  [chain.mainnet, chain.polygon],
  [defaultProvider()],
)
```

<Callout emoji="⚠️">
  Only having `defaultProvider` in your API providers will make the chain use
  the public RPC URL which could lead to rate-limiting. It is recommended to
  also include another API provider in your list (such as: `alchemyProvider`,
  `infuraProvider` or `staticJsonRpcProvider`).
</Callout>

## Return Value

```ts
{
  chains: Chain[],
  provider: BaseProvider,
  webSocketProvider: WebSocketProvider
}
```
